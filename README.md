
# Ainun
Ainun want to bundle that cakes and apples into boxes and give them to her friends


## Prerequisite
* Golang >= 1.19


## Installation
Gunakan package manager [go](https://go.dev/) untuk install dependency
```bash
go get
```


## Usage
### Run app
```bash
go run main.go
```

### Testing
```bash
go test
```
